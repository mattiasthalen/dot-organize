# dot-organize

Validate and create HOOK manifests for data modeling.

## Installation

```bash
pip install dot-organize
```

## Quick Start

```bash
# Validate a manifest
dot validate manifest.yaml

# Create a new manifest interactively
dot init

# View example manifests
dot examples list
dot examples show minimal
```
